# Hostel Management Desktop Application with Tkinter

A desktop-based hostel management system developed using **Python** and **Tkinter** to simplify hostel operations such as student registration, room allocation, vacancy tracking, and payment management.

## Overview

This application is designed to help hostel administrators manage hostel activities in an efficient and organized way. It provides a simple graphical user interface where users can register students, manage rooms, allocate and deallocate rooms, track vacancies, and handle payment records.

The system is suitable for academic purposes, learning desktop application development, and demonstrating the practical use of **Python OOP**, **Tkinter GUI**, and **database integration**.

## Features

- Student registration and management
- Hostel and room management
- Room allocation and deallocation
- Vacancy tracking
- Payment recording and monitoring
- User-friendly desktop interface built with Tkinter
- Organized project structure using models and database connection

## Technologies Used

- **Python**
- **Tkinter**
- **SQLite** 
- **Object-Oriented Programming (OOP)**
